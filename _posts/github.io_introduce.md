
### 🎉 我的博客结构与用途简介

```text
guobosun.github.io/
├── _config.yml        # 博客主配置文件
├── _posts/            # 博客文章目录
│   └── *.md           # 每一篇博客文章（以日期命名）
├── index.md           # 首页内容
├── about.md           # 关于我页面
├── assets/            # 存放图片、CSS、JS等资源
├── _layouts/          # 页面布局模板
├── _includes/         # 复用组件（页眉、页脚等）
├── _site/             # 本地编译输出目录（不上传）
└── README.md          # 仓库说明文件
